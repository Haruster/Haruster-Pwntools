# Python context.arch

- context.arch는 pwntoolsdml 구조설정을 하며, 기본값으로는 i386이 들어가 있다.

## arch의 설정값으로는 다음이 들어갈 수 있다.

- aarch64 
- alpha
- amd64
- arm
- avr
- cris
- i386
- ia64
- m68k
- mips
- mips64
- msp430
- none
- powerpc
- powerpc64
- s390
- sparc
- sparc64
- thumb
- vax
- ppc64
- ppc
- x86_64
- x86
- i686
- arm7l
- armeabi
- arm64

## 설정 방법은 다음과 같습니다.
 context.arch = 'arch설정값' #예) arch64
